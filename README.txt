  This source code provide the solution of thoughtWorks' Coding Assignment:
  Problem Three: Merchant's Guide to the Galaxy.


  It should be run on environment preloaded JDK 1.6 or higher version.

  The input file's position is "galaxyMerchant/src/resources".
  The output file will be placed in root directory of "E" driver according to
  configuration in  class Constant.

  Quick start:

  Please import the source code into IDE as a maven project, then call the main method in class
  MerchantBusinessCaller.
